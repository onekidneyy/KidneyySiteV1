Super S Website

This website was built using react.js for a Solana meme coin called $SuperS.

The main project goal is nostalgia so when working out design ideas, I was thinking what more nostalgic than the classic windows style.

The website design host many cool features, the apps are built like a folder system and you can open multiple at a time it also has an arcade-style game for the community to compete against each other for the high score.

Visit www.superssolana.com to try it out!!

APP Features

Animations

App holding all the animations done by NickGoonz

Twitter

App that takes you to the official SuperS twitter

Telegram

App that takes you to the official SuperS telegram

SuperJump

Crypto arcade game inspired by the classic "Doodle Jump" game

Memes

App holding all the memes done by the SuperS community

Instagram

App that takes you to the official SuperS Instagram

RoadMap

App holding the roadmap showcasing future plans for SuperS

Jeet

App that takes you to Sol Incinerator to burn your SuperS

Jupiter Exchange

App that takes you to Jupiter where you can buy/sell SuperS
